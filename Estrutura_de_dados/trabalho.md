# Trabalho valendo 3.0 pontos

Implemente um mini-cadastro de itens de uma taberna.
Cada item é **COMIDA** (guarda `kcal`) ou **BEBIDA** (guarda `teor_alcoolico`).
Guarde tudo num **vetor fixo** `Item vet[MAX]` (estrutura homogênea).
Faça funções para **inserir**, **listar** e **buscar por id**.
Use **enum + union** (estrutura heterogênea) e **ponteiros** nas funções.
